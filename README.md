# Folkline3 (React)


Proiect React (variantă web) reconstruit din aplicația originală. Conceput să funcționeze 100% pe GitHub Pages.


## Pași rapizi


1. `npm install`
2. `npm start` — rulează local (dezvoltare)
3. `npm run build`
4. `npm run deploy` — publică pe GitHub Pages (homepage setat în package.json)


Linkul final recomandat:
`https://sabrinagiulia2008-byte.github.io/folkline3/`


---


*Asigură-te că ai creat repository-ul `folkline3` pe contul tău GitHub și l-ai legat local înainte de `npm run deploy`.*
